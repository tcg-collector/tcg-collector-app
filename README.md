# TCG Collector

App para colecionadores de cartas físicas de Pokémon TCG.

Gerencie sua coleção, acompanhe o valor do seu portfólio e descubra cartas que faltam.

---

## Stack

- **App (mobile + web):** React Native + Expo
- **Backend:** Node.js + Express
- **Banco de dados:** MongoDB Atlas
- **API de cartas:** [PokéTCG API](https://pokemontcg.io)
